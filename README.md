# `insomnia`

This library provides a cross-platform interface for inhibiting power management operations.

## Features

| Platform                     | Automatic suspend | Manual suspend | Manual shutdown |
| ---------------------------- | ----------------- | -------------- | --------------- |
| Linux (via `systemd-logind`) | ✓                 | ✓              | ✓               |
| Windows                      | ✓                 | ✓              |                 |
